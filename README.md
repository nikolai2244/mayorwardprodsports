# MayorWard ProdSports

NFL AI Simulator with live odds integration.

## Features

- Live NFL game data from ESPN API
- Real-time betting odds from The Odds API
- Game simulation engine with team strength modeling
- Interactive UI with live matchup displays
- Parlay prediction capabilities

## Deployment

Deployed on Vercel: [mayorwardprodsports.vercel.app](https://mayorwardprodsports.vercel.app)
